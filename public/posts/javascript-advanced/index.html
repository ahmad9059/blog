Handling Errors with `try…catch#

The try...catch block allows you to handle errors gracefully. The try...catch statement is used to handle errors, and throw is used to create custom errors.

Basic Example of try...catch

try {
    let x = y + 5;  // âŒ ReferenceError: y is not defined
} catch (error) {
    console.log("An error occurred:", error.message);
}

    if (b === 0) {
        throw new Error("Cannot divide by zero!");  // ğŸš€ Custom error
    }
    return a / b;
}

    console.log(divide(10, 0)); // âŒ Throws an error
} catch (error) {
    console.log("Error:", error.message);
}

finally Block (Always Executes)#

    console.log("Trying something...");
    throw new Error("Something went wrong!");

    console.log("Caught Error:", error.message);

    console.log("This runs no matter what!");
}
